-----------------Back-end------------------
`cd backend`

Install dependencies

  `npm install`

Start microservices

  npx nodemon authenticate.service.js --port 5001
  npx nodemon carts.service.js --port 5002
  npx nodemon wishlist.service.js --port 5003
  npx nodemon products.service.js --port 5004
  npx nodemon categories.service.js --port 5005



------------------------Front-end-----------------
cd ui

npm install

ng serve
